----
Note
----

A modular Telegram Userbot running on Python3 with sqlalchemy database.

Based on `ProjectBish <https://github.com/adekmaulana/ProjectBish>`_ and `WeebProject <https://github.com/BianSepang/WeebProject>`_ Userbot.
